Физкультурные группы (оптимально: чел.)

  • Для добавления подгруппы в состав ФГ перетащите её мышью из списка слева;
  • Для редактирования состава ФГ кликните по иконке — входящие в состав подгруппы выделятся синим цветом;
  • Щелкните по выделенной подгруппе, чтобы удалить её из состава ФГ;
  • Для выхода из режима редактирования ещё раз щёлкните по иконке ;
  • Для удаления физкультурной группы щёлкните по иконке
    ВНИМАНИЕ! Восстановить удалённую группу будет нельзя.
  • Для изменения названия ФГ кликните по нему и введите новое в появившемся поле
  • Для окончательного сохранения созданной ФГ нажмите на иконку
