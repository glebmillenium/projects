Преподаватели

  • Выберите преподавателя из выпадающего списка;
  • Для привязки к выбранному преподавателю физкультурной группы перетащите мышью соответствующую группу из списка слева в блок с именем преподавателя;
  • Для редактирования распределения щёлкните по иконке — добавленные физкультурные группы выделятся синим цветом. Для удаления группы из распределения просто кликните по ней.
