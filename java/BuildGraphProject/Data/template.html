Описание

Приложение предназначено для проведения научных, учебных не коммерческих исследований в области математики, геометрии, физики и др. фундаментально-классических дисциплин с целью визуального показа или выявления некоторых математических зависимостей при построении графика или нескольких графиков функций в двумерной системе координат с рекурсивной вложенностью каждой функции не более 255 единиц.

Начало построения

  • Ввод нескольких графиков

    • Для построения нескольких функций (двух и более) в приложении имеется кнопка "Добавить функцию" которая добавит область ввода для новой функции

  • Удаление графика

    • Для удаления (уменьшении числа графиков) используется кнопка "Удалить график"

  • Изменение цвета построения кривой линии

    • При построении нескольких графиков функций удобнее использовать уникальную раскраску для каждой кривой. В палитре пользователь может выбрать до 16581375 различных цветов.

  • Масштабирование

    • Для увеличения и уменьшения расстояния между делениями осей имеется функция масштабирования системы построения графиков. По умолчанию значение выставлено на x1, что приблизительно равно 40 пикселям на экране монитора

  • Сдвиги осей координат

    • Чаще всего при научных исследованиях требуется провести динамический обзор построенных функций ввиду их ограниченной области значения и определения. Для этого в приложении имеются кнопки сдвига координат как вдоль оси абсцисс, так и вдоль оси ординат

  • Выделение десятичных координат

    • При значительном увеличении масштаба осей координат (как в сторону оси абсцисс, так и в сторону осей координат), теряется точность измерения графиков, ввиду не достаточно точного деления осей координат. Поэтому при необходимости пользователь может установить разрисовку десятичных долей координат функции.

  • Построение

    • Для построения введенных функций в виде формул зависящих от одной переменной x, с помощью кнопки "Построить" мы получаем готовую разрисовку введенной функции

Поддерживаемые математические модели

  • Типы функций

    • Программа расчитана на построение одноместных* функций. Заданных через временной диапазон (y = f(t)), через систему координах осей xOy, параметрически заданных функций y(t) = x(t) и заданных на полярной оси координат.

    *Прим. Одноместная функция - функция зависимости одной переменной: f(x), f(t), но не f(x1,x2..xn)

  • Константы

    • Math.E
    Константа Эйлера, основание натурального логарифма. Приблизительно 2.718.
    Math.LN2
    Натуральный логарифм 2, приблизительно 0.693.
    Math.LOG2E
    Логарифм E по основанию 2, приблизительно 1.442
    Math.LOG10E
    Логарифм E по основанию 10, приблизительно 0.434
    Math.PI
    Отношение длины окружности к ее диаметру, приблизительно 3.14159.
    Math.SQRT12
    Квадратный корень из 1/2; иначе говоря, 1 поделить на корень из 2, приблизительно 0.707.
    Math.SQRT2
    Квадратный корень из 2, приблизительно 1.414.
    Math.LN10
    Натуральный логарифм 10, приблизительно 2.302.

  • Арифметические(линейные) операции

    • Сложение (обозначается символом плюса «+») — это одна из четырёх элементарных математических операций арифметики, вместе с вычитанием, умножением и делением. Сложение двух натуральных чисел есть общая сумма этих величин.

    Вычитание (убавление) — одна из четырёх арифметических операций; операция, обратная сложению. Обозначается знаком минус «−».

    Умножение (обозначается символом «*») — одно из четырёх основных арифметических действий, бинарная математическая операция сложения указанного количества одинаковых слагаемых.

    Деление (операция деления) — действие, обратное умножению. Деление обозначается косой чертой /.

  • Не линейные математические операции

    • Полиномы

      • Полином (он же многочлен) - сумма одночленов.
      В частности, многочлен от одной переменной есть конечная формальная сумма вида: c0+c1x1+..+cmxm; где c0..m - фиксированные коэф., x - переменная. В BuildGraphProject - для возведения в степень используется функция возведения в степень (показательная): pow(x,c)

    • Логарифмические, показательные

      • pow(x, a) - возведение числа в степень
      Синтаксис
      pow(x, a) = xa
      Пример:
      pow(x,3) = x3, при х=2, 23 = 8

      sqr(x) - возведение числа в квадрат
      Синтаксис
      sqr(x) = x2
      Пример:
      sqr(4) = 42 = 16

      cbr(x) - возведение числа в куб
      Синтаксис
      cbr(x) = x3
      Пример:
      cbr(4) = 43 = 64
      sqrt(x) - извлечение квадратного корня из x
      Пример:
      sqrt(9) = 3

      ln(x) - возвращает натуральный (по основанию E) логарифм числа Синтаксис

      ln(x) = lne x
      Для нахождения логарифма по другому основанию используйте следующий синтаксис: loga b (логарифм b по основанию a) = ln(b)/ln(a)
      Пример: ln(3) = lne 3 ~ 1,098

    • Тригонометрические функции

      • sin(x) – возвращает синус аргумента x
      cos(x) – возвращает косинус аргумента x
      tg(x) – возвращает тангенс аргумента x
      ctg(x) - возвращает котангенс аргумента x
      Пример:
      cos(0) = 1
      sin(pi/3) ~ 0,866

      Угол, который в качестве аргумента принимают эти функции задаётся в радианах, а не в градусах.
      Для того, чтобы перевести значение угла из градусов в радианы
      необходимо умножить его на PI и разделить на 180. Также в BGP имеется метка, которая указывает на представление единицы измерения в градусах "_gr". Например cos(60°) ~ cos(60_gr)

    • Обратные тригонометрические функции

      • arccos – возвращает арккосинус от числа т.е. такой угол (в радианах), косинус которого равен аргументу
      arcsin – возвращает арксинус от числа т.е. такой угол (в радианах), синус которого равен аргументу
      arctg – возвращает арктангенс от числа т.е. такой угол (в радианах), тангенс которого равен аргументу
      arctg2 – возвращает числовое значение между - π и π, представляющее собой угол Theta для точки (x,y). Это угол, отсчитываемый против часовой стрелки и измеряемый в радианах, между положительным лучом оси X и точкой (x,y). Заметим, что порядок аргументов у этой функции такой, что координата по Y передается первой, а по X - второй.
      sc(x) – возвращает секанс аргумента x
      csc(x) – возвращает косеканс аргумента x
      Пример: arccos(1/2) = π/3;

  • Поддержка особых функций, предоставляемых средствами функционально-логического программирования

    • Поиск максимума функций

      • Math.max([value1[,value2[, ...]]]) - возвращает наибольший из списка аргументов.
      Аргументы
      value1, value2, ... - числа
      Пример:
      Math.max(1, 5, 4, 2) = 5

    • Поиск минимума функций

      • Math.min([value1[,value2[, ...]]]) - возвращает наименьший из списка аргументов.
      Аргументы
      value1, value2, ... - числа
      Пример:
      Math.min(1, 5, 4, 2) = 1

    • Функция знака

      • sign(x) - возвращает знак числа, указывающий на то, является ли x отрицательным, положительным или нулём, применяется как замена единичной функции Хевисайда(функция единичного скачка) - δ(t)
      Пример:
      sign(3) = 1
      sign(-3) = -1
      sign(0) = 0

    • Функция Хевисайда

      • Hvs(x) - Функции Хевисайда(функция единичного скачка) - δ(t)
      Пример:
      Hvs(3) = 1
      Hvs(-3) = 0
      Hvs(0) = 0

    • Округление аргумента

      • Получение наибольшего целого аргумента

        • round(x) - Если дробная часть числа x равна 0.5 или больше, то аргумент округляется к следующему большему целому. Если дробная часть меньше, чем 0.5, то округление производится до ближайшего меньшего целого.

      • Получение наименьшего целого аргумента

        • floor(x) - Возвращает наибольшее целое, меньшее или равное аргументу

    • Генерация случайного числа

      • random() - генерация случайного числа в диапазоне от [0;1)
      random() * (max - min) + min - генерация случайного числа в диапазоне от [min;max), где min,max - числа и задаются самим пользователем.
